# ProyectoFinal
En el espacio nombre la persona debe escribir su nombre completo.
Identificación: En ese cuadro se escribe el número de identificación, cedula o documento personal.
Teléfono: Aquí se coloca un número de teléfono.

Origen: Se escribe el lugar de donde la persona va a salir.
Destino: Es el lugar al que la persona desea viajar.
Clase: Aquí se escoge el tipo de servicio, ya sea económico o ejecutivo.

Después de llenar toda la información se debe presionar el botón RESERVAR.
Al presionar este botón el programa revisa que todos los datos estén llenos, toma la información personal, lee el origen, destino y clase seleccionados, y crea la reserva con esos datos.

El cuadro grande sirve para mostrar mensajes del programa.
Confirmación de que la reserva se realizó de manera correcta.
Información sobre el viaje.
Mensaje de error si falta algún dato
Detalles de la compra o el valor del tiquete.
